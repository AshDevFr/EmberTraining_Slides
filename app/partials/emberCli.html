Ember-cli

  • The Ember.js command line utility

  • Provides a Broccoli powered asset pipeline,
  • a strong conventional project structure,
  • and a powerful addon system for extension

Install with npm i -g ember-cli

Create a new project

ember new <project name>

The project structure

The app structure

Add a new resource

ember generate resource <resource name> <options>

Install a plugin

ember install <plugin name>

Add a bower dependency

Exemple: Install a localstorage adapter 


   $ bower install ember-localstorage-adapter

In the file ember-cli-build.js 


   var EmberApp = require('ember-cli/lib/broccoli/ember-app');

   module.exports = function(defaults) {
      var app = new EmberApp(defaults, {});

      app.import(app.bowerDirectory +
            '/ember-localstorage-adapter/localstorage_adapter.js');

      return app.toTree();
   };
